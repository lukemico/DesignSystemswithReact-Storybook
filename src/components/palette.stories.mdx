## Overview
Like in other ui in mondays interface, typography works by principle of accessibility before aesthetics.
Therefore the text should be readable and help the user understand what’s important by well contrasted size and colors hierarchy.

## Colours by category

<div className="columns">
	<div className="tokens__list">
		<ul>
			{Object.keys(palette).map((colourCategory, idx) => {
				const colours = Object.keys(palette[colourCategory]).map(
					(colour, idx) => {
						return (
							<div className="tokens__list__row" key={idx}>
								<div className="tokens__list__value">
									<div
										className="tokens__list__value__swatch"
										style={{
											backgroundColor:
												palette[colourCategory][colour],
										}}
									/>
								</div>
								<span>
									<pre>{`palette.${colourCategory}[${colour}]`}</pre>
								</span>
							</div>
						);
					},
				);
				const openFirst = idx === 0;
				return (
					<li className="tokens__list__dropdown" key={idx}>
						<details open={openFirst}>
							<summary className="tokens__list__category">
								{colourCategory}
							</summary>
							<div>{colours}</div>
						</details>
					</li>
				);
			})}
		</ul>
	</div>
</div>
